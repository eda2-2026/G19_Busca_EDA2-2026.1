## Descrição do Projeto

Essa aplicação permite o usuário consultar uma base de dados que contém os filmes de melhor avaliação do IMDB. O objetivo do projeto é estudar e aplicar algoritmos de busca em uma situação útil, onde nesse projeto o usuário pode pesquisar pelos filmes através de filtros como gênero ou ano e buscar através do nome, utilizando algoritmos de **Busca Sequencial** e **Tabela Hash**.

## Como Rodar

### Pré-Requisitos

* Python 3.x
* Pip

### Instalação

1. Clonar o repositório
   
```bash
git clone https://github.com/eda2-2026/G19_Busca_EDA2-2026.1
cd G19_Busca_EDA2-2026.1
```

2. Instalar dependências

```bash
pip install flask
```

3. Executar

```bash
python app.py
```

4. Acessar no navegador: http://127.0.0.1:5000
